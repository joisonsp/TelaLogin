Rodando o back-end:
- Abrir a pasta "back-end" no seu terminal;
- Digitar "npm install"
- Digitar "node index"

Executando a parte front
-Digite "joison" no campo de login;
-Digite "netflix" no campo de senha;
-Clique no botão "log in";

OBS: Perceba que no terminal onde o servidor está rodando aparece o login e senha válidos.
OBS2: Observa-se que com o login válido, a API retorna um token de autenticação, se tentar com um login invalido esse token não é gerado.
